---

# 💻 Frontend Technologies

| Technology | Purpose | Why Selected |
|------------|----------|--------------|
| React.js | Build interactive user interface | Component-based architecture, reusable UI components |
| Vite | Frontend build tool | Faster development server and optimized build performance |
| Axios | API communication | Simpler and cleaner HTTP requests than Fetch API |
| React Markdown | Render AI reports | Displays formatted AI-generated markdown content |

---

# ⚙️ Backend Technologies

| Technology | Purpose | Why Selected |
|------------|----------|--------------|
| Python | Backend programming language | Easy integration with AI libraries and rapid development |
| FastAPI | REST API framework | High performance, automatic API documentation, modern architecture |
| Google Gemini API | AI report generation | Generates structured market research reports |
| python-dotenv | Environment variables | Secure API key management |

# 🤔 Why These Technologies?

## Why React?

- Reusable Components
- Fast UI Rendering
- Large Ecosystem
- Industry Standard

---

## Why FastAPI instead of Flask?

- Better Performance
- Automatic API Documentation
- Async Support
- Modern Python Framework

---

## Why Gemini AI?

- High-quality text generation
- Easy API integration
- Suitable for business report generation

---

## Why Axios instead of Fetch API?

- Cleaner syntax
- Better error handling
- Automatic JSON parsing
- Request configuration
